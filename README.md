# FUTURE_DS_01
Sales Performance Analysis Dashboard

1. Overview
This project analyzes retail sales data using Python and Power BI to identify sales trends, regional performance, product performance, and business KPIs.

2. Tools Used
-Python
- Pandas
- Jupyter Notebook
- Power BI

3. Project Workflow
* Data Cleaning
* Exploratory Data Analysis
* KPI Calculations
* Dashboard Development

4. Dashboard Pages
* Executive Summary
* Product Analysis
* Regional Analysis
* Trend Analysis

5. Key KPIs
* Total Sales
* Total Profit
* Profit Margin
* Total Orders
* Quantity Sold
